● 두 가지 방법이 있습니다:

  ---
  방법 1: 앱 사이드바에서 직접 입력 (간편)

  앱 실행 후 왼쪽 사이드바 "🔑 API Key" 섹션에 입력하면 됩니다.
  선택한 모델에 따라 자동으로 해당 프로바이더 키 입력란이 표시됩니다.

  ┌──────────────────────┬───────────────────┐
  │         모델         │     필요한 키     │
  ├──────────────────────┼───────────────────┤
  │ Claude (Anthropic)   │ Anthropic API Key │
  ├──────────────────────┼───────────────────┤
  │ GPT-4o (OpenAI)      │ OpenAI API Key    │
  ├──────────────────────┼───────────────────┤
  │ Gemini (Google)      │ Google API Key    │
  ├──────────────────────┼───────────────────┤
  │ Groq Llama / Mixtral │ Groq API Key      │
  └──────────────────────┴───────────────────┘

  ---
  방법 2: 환경변수로 설정 (자동 로드)

  터미널에서 미리 설정하면 앱 실행 시 자동으로 입력됩니다:

  # Windows (PowerShell)
  $env:ANTHROPIC_API_KEY="sk-ant-..."
  $env:OPENAI_API_KEY="sk-..."
  $env:GOOGLE_API_KEY="AIza..."
  $env:GROQ_API_KEY="gsk_..."

  streamlit run app.py

  또는 .env 파일을 프로젝트 루트에 만들어 사용할 수도 있습니다 (별도 라이브러리
  추가 필요).

  ---
  권장: 개발 중에는 사이드바 입력, 배포 시에는 환경변수 사용을 추천합니다.
